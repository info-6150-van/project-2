## Note 1

Note that the student name is going to be removed from the log entries due to following considerations:

- The project is no longer group based and is intended to be individual-based, meaning the group name is unassigned and irrelevant.
- The project is handed via GitHub using pull requests, however the original repo is **public**, meaning there is a risk of PII leakage if the students' real names are present (potentially in conflict with the NEU privacy and security policies).

To be consistent with the logging policy, the github user name will be provided instead.

## Note 2

AI usage that is irrelevant to the project (for example, for another class or for a standalone personal project) won't be logged in the document.
